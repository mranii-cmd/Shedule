💭 Gestion des Souhaits des Enseignants

📥 Importer les Souhaits depuis Excel

Dans l'onglet Rapports & Export, importez les souhaits de vos enseignants depuis un fichier Excel. Le fichier doit contenir les colonnes : Enseignant, Choix1, C1, TD1, TP1, Choix2, C2, TD2, TP2, Choix3, C3, TD3, TP3, Contraintes.

📖 Instructions

  1. Téléchargez le template Excel en cliquant sur "Télécharger Template Excel"
  2. Remplissez le fichier avec les souhaits de chaque enseignant
  3. Pour chaque choix, indiquez le nombre de séances souhaitées (0 = refus, vide = flexible)
  4. Cliquez sur "Importer Souhaits" et sélectionnez votre fichier rempli

📋 Format du Fichier Excel
